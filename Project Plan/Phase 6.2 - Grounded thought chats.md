# Phase 6.2: Grounded Thought Chats & Actions Promoter

This specification details details of the AI dialogue grounding system, brainstorm flow mechanics, and how users can promote tasks or decisions directly from conversational sessions.

## 🗺️ Progression Targets

### 1. Context Grounding API
- Ground active user dialogue prompts using parent thought metadata, custom category types, and tag parameters.

### 2. Conversational Action/Task promotion
- Expose buttons in J.A.R.V.I.S. markdown listings to directly create a task or action item from chat responses.
- Implement `/api/action-items` triggers using structured chat content.
